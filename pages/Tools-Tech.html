Tools and Technologies

For our Home Aquaponics Monitoring System (H.A.M.S) there will be two forms of hardware that are required outside the basic laptop/desktop computer which most people have access too. One will be a Raspberry Pi device and the other temperature sensors, which are very cheap and obtainable through many online stores. These two will be the basis for allowing us to create, with further software implementations, a basic set up for monitoring environment variables.

Alongside the hardware components we will require software and tools such as listed below.
  • Raspbian OS
  • Python
  • RRDTool
  • Git
  • Chart.js Library
  • Linux Distribution - Debian
  • SSH Protocol
  • Nginx
  • Git
Our github audit trail somewhat represents the amount of work conributed by each team member, though majority of our team work was done using other collaborative tools such as Microsoft Teams. We have attempted to collaborate, each commiting our own changces, though some team members worked more on other parts of the project such a the concept pitch video, which is note represented in github.

All of these tools/resources/software's are open source and readily available to anyone making them ideal for us to use in our product/project without the need for purchasing anything.

There will be no software licenses needed at present in order to get our H.A.M.S into the initial set up.

Our group as a whole have had some prior experience in the tools and software listed such as Python, but many of us have not had to deal with software and tools such as RRDTool or Nagios. Through the setup of H.A.M.S, Sam has created, implemented and got the hardware and software set up through the use of a Raspberry Pi device. From this, team members have been able to learn along the way and research topics that we were newly introduced too, expanding our knowledge of devices, software and hardware in relation to our project.

Tools

Tools Used

  • GitHub
  • Grammarly
  • Microsoft Teams
  • Microsoft OneNote
  • Microsoft Word
  • TeXworks
  • Textadept

GitHub

GitHub was used for the groups Git repository and to host the group website, GitHub is easy to use and the GitHub pages feature makes it easy to host a website.

Grammarly

Grammarly was used to help with spell checking and sentence structures, after problems with nonsense suggestions, it was not used anymore.

Microsoft Teams

Microsoft Teams (Teams for short) was the main form of communication used for this assessment, Teams made it easy for group members to share files and the extra features, like the calendar, made it easy to plan meetings in advanced.

Microsoft OneNote

Microsoft OneNote (OneNote for short) was the main program used for writing, OneNote made it easy for group members to collaborate on a piece of writing. The tab and page features in OneNote made it easy to organize the writing sections in our assessment.

Microsoft Word

Microsoft Word (Word for short) was initially used for writing the report, after some discussion, it was agreed that the document would be written in LaTeX.

TeXworks

TeXworks was used to write the document in LaTeX, it is a simple, multi-platform front-end program that makes writing in LaTeX easy.

Textadept

Textadept is a fast, minimalist, and remarkably extensible cross-platform text editor.
